Cabinet21 (پوشه اصلی پروژه)
├── app/
│   ├── layout.js     (قالب کلی سایت مثل هدر و فوتر)
│   └── page.js       (صفحه اصلی سایت شما - همان صفحه اول)
├── public/           (محل قرارگیری عکس‌ها، لوگو و فونت‌ها)
├── package.json      (لیست ابزارها و پکیج‌های نصب شده)
├── next.config.js    (تنظیمات خود نکست‌جی‌اس)
└── jsconfig.json     (یا tsconfig.json برای تنظیمات مسیردهی کدهایتان)

ران کردن پروژه
npm run dev
